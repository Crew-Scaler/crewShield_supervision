### FedRAMP 20x Modernization
- Shift from 800-53 controls to Key Security Indicators (KSIs)
- Automated compliance validation
- Machine-readable evidence requirements
- Continuous monitoring emphasis
- Target: "weeks instead of months/years" for authorization

### EU AI Act Implementation
- Phased application: Feb 2025, Aug 2025, Aug 2026
- High-risk system conformity assessment requirements
