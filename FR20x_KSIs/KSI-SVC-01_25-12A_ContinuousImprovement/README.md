# KSI-SVC-01: Continuous Improvement

## Overview

This folder contains documentation and resources for FedRAMP 20x Key Security Indicator **KSI-SVC-01**.

For complete information about all FedRAMP KSIs, please refer to the official JSON specification:
- [FedRAMP Key Security Indicators (KSI) JSON](https://github.com/FedRAMP/docs/blob/main/FRMR.KSI.key-security-indicators.json)

- [NIST SP 800-53A](https://csrc.nist.gov/pubs/sp/800/53/a/r5/final)
