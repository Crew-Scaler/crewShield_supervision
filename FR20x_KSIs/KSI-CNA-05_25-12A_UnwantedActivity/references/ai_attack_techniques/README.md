## Key Findings

### Attack Landscape
- **Adaptive RL attacks** now bypass ML-based IDS in real-time
- **Economic DoS (EDoS)** exploits cloud auto-scaling for cost amplification
- **Federated botnets** coordinate attacks across distributed nodes
- **Legitimacy mimicry** makes attack traffic indistinguishable from legitimate users
- **Multi-vector coordination** simultaneously targets L3/L4/L7

### Defense Evolution
- **Federated learning** enables privacy-preserving collaborative defense
- **Programmable data planes** (P4/SmartNICs) provide high-speed mitigation
- **LLM integration** offers semantic attack understanding
- **Graph neural networks** excel at botnet topology analysis
- **Hybrid multi-layer systems** required for comprehensive protection

### Technology Stack
- **Attack Tools:** TensorFlow/PyTorch RL, CleverHans, adversarial frameworks
- **Defense Tools:** PyTorch Geometric, PySyft, P4 switches, SmartNICs
- **Datasets:** CIC-DDoS2019, UNSW-NB15, CTU-13, 5G-NIDD
- **Environments:** SDN/NFV, cloud/serverless, 5G/O-RAN, IoT/edge
