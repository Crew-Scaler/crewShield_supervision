## Key Technologies Mentioned

### Cryptographic Techniques
- Zero-Knowledge Proofs (ZKPs): ZKAUDIT, zkDL, zkReLU
- Proof-of-Training (PoT): Ownership verification
- Trusted Execution Environments (TEE): Attestable audits
- Cryptographic Signatures: Sigstore, model signing

### Standards and Frameworks
- W3C PROV: Data provenance standard
- PROV-JSON: Machine-readable provenance
- ProvML: ML-specific provenance extension
- SPDX: ML-BOM working group
- CycloneDX: AI-BOM standardization

### Tools and Platforms
- MLflow: Model registry and experiment tracking
- yProv4ML: Provenance tracking library
- Hugging Face: 2M+ model repository
- DVC: Data version control
- MCP Registry: Agent metadata publishing

### Detection Methods
- STRIP: Backdoor detection via decision values
- Neural Cleanse: Optimization-based backdoor discovery
- Cluster Impurity (CI): Latent space clustering
- SAM: Sharpness Aware Minimization
- SynthID: Google's watermarking system
